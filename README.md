#  Phonebook Manager

A simple and fully functional **Phonebook web application** built with **PHP, MySQL, HTML, CSS, and Bootstrap**.  
This app allows users to **add, edit, delete, and search contacts** with optional profile photos. It also includes **pagination**, **form validation**, and a **responsive UI** that works on all devices.

## Features

- ✅ Add new contacts with name, phone, email, and optional photo
- ✅ Edit and update existing contacts  
- ✅ Delete contacts safely with confirmation  
- ✅ Search contacts by name, phone, or email  
- ✅ Pagination for large lists of contacts  
- ✅ Bootstrap responsive UI for mobile and desktop  
- ✅ Form validation for phone numbers and required fields  
- ✅ Optional default image if no photo is uploaded  

---

Technologies Used

- Frontend: HTML, CSS, Bootstrap 5  
- Backend: PHP  
- Database: MySQL / MariaDB  
- Server: XAMPP / Localhost
